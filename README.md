# OBBBA_Individual-Provisions 2026
OBBBA Individual Provisions "What-Changed" Dashboard (Power BI/Excel). Encode 2026 OBBBA individual changes (Standard deduction, Tip deduction, overtime deduction, SALT cap, Car -loan-interest), as a rule table, then model tax impact across sample taxpayer profiles with phase-outs. Shows rule-application + scenario modeling.
